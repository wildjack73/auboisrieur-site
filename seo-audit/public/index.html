Mes mini-audits

    Lexique — c'est quoi tous ces termes ?
    • Top 3 / local pack : les 3 fiches qui apparaissent avec la carte quand on cherche « métier + ville » sur Google. Le but de l'audit : voir ce qu'ont en commun ces 3 fiches partout en France, pour faire pareil.
    • Catégorie principale / secondaire : la catégorie d'activité de la fiche Google (ex. « Salon de coiffure »). On peut en ajouter d'autres en secondaires. C'est l'un des facteurs les plus importants.
    • Description : le texte « de la part du propriétaire » sur la fiche. L'audit dit combien de caractères/mots écrire et quels mots-clés y mettre.
    • Avis (chips Google / « place topics ») : les petites étiquettes sous « Avis » (ex. « gentillesse 23 »). Ce sont les mots que Google repère dans les avis des clients. À reprendre dans vos réponses et à suggérer à vos clients.
    • Citations / NAP : les annuaires où votre établissement est listé (PagesJaunes, etc.). Plus vous y êtes (avec les mêmes nom/adresse/téléphone), mieux c'est.
    • Domain Rank / trust / backlinks / domaines référents : la « popularité » d'un site web aux yeux de Google, basée sur les liens qu'il reçoit d'autres sites. Sur 1000 pour le Domain Rank.
    • Volume de recherche : combien de fois par mois les gens tapent cette recherche sur Google.
    • Allintitle / KGR : indicateurs de concurrence sur un mot-clé (combien de pages le ciblent dans leur titre).
    • Vision / objets / labels : analyse automatique des photos pour savoir ce qu'elles montrent (devanture, équipe, produit…) → quoi photographier en priorité.
    • Posts Google : les publications (actus, offres) qu'on peut poster sur sa fiche, comme sur un réseau social.
